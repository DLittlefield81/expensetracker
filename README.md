## Installation

Follow these steps to install this project on your local machine.
- Open Bash editor
- mkdir reactportfolio
- cd reactportfolio
- git clone https://github.com/DLittlefield81/ExpenseTracker.git
- npm install

## Usage
- After cloning this repository...
- Open bash terminal 
- npm run start
